# Rock Paper Scissors

#### Play a two player game of rock, paper, scissors, February 10/2016

#### By Abby Rolling & Kevin Mattison

## Description

This web app allows two players to play a round of rock, paper, scissors and returns the result.

## Technologies Used

* Java
* Fluentlenium
* Spark
* HTML
* Gradle
